# Quantum Mechanics Notes

Project: Quantum-Mechanics-Notes
Author: [CriticalLine](https://github.com/CriticalLine)

## Overview

Concise, structured notes and derivations for core quantum mechanics topics. Intended as a personal reference and teaching aid.

## Scope

- Foundations: postulates, wavefunctions, operators
- Formalism: Hilbert spaces, bras/kets, spectral theorem
- Dynamics: Schrödinger, Heisenberg, interaction pictures
- Applications: harmonic oscillator, hydrogen atom, scattering, perturbation theory
- Advanced topics: identical particles, quantum information (optional)
